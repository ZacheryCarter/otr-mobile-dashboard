# OTR Mobile Dashboard Prototype

Work-in-progress Python/Tkinter application for an internal-style OTR Mobile dashboard prototype.

## Current Features
- Login/signup flow
- Email validation
- Role selection menu
- Role-based permissions
- Tkinter UI foundation
- OTR Mobile logo/image assets included

## Project Goal
This project is being built as a learning and portfolio project to simulate a dashboard system with user authentication, role handling, and a desktop GUI.

## Planned Features
- Existing user login checks
- Persistent user storage
- Password handling
- Improved role management
- Expanded Tkinter dashboard screens
- Better UI structure and navigation

## Tech Used
- Python
- Tkinter
